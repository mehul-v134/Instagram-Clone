# Instagram-Clone
Instagram Clone using Flutter & Firebase

# Features
* Feed
* Search & Explore
* Post
  * Like and Comment on Posts
* Activity
  * Show interactions (likes) on your posts
* Profile
  * See followers & following
* Change Theme
